# 🧠 SchizoWellness: AI-Driven Mental Health Support Platform

## 📚 Overview
**SchizoWellness** is a web-based platform designed to support individuals living with schizophrenia.  
It offers **🤖 AI-powered chatbot interactions**, **📝 self-assessment tools**, and **📖 educational resources**, providing a safe, accessible, and confidential environment for users seeking mental health assistance.

---

## ✨ Key Features
- 🤖 **AI Chatbot Assistance**  
  Immediate, empathetic support using JavaScript-based logic and optional external APIs.

- 📝 **Self-Assessment Tools**  
  Simple, user-friendly questionnaires to evaluate symptoms and promote self-awareness.

- 📖 **Educational Resources**  
  Curated articles, videos, and helpline links focused on schizophrenia and mental well-being.

- ⚡ **Frontend-Only Architecture**  
  No database or backend — all functionalities are managed purely on the frontend.
